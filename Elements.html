Elements

What is an Element?

Chemical elements are pure substances that consist of atoms with the same number of protons. The number of protons in an atom make each atom a unique element. So while different elements can have the same number of neutrons or electrons, they cannot have the same number of protons.

Different types of the same Element?

Elements in nature can be found having different numbers of neutrons and thereby, a different atomic mass, while still being the same element. The same element with a different mass is known to be an isotope of the element, which plays a part in finding the average atomic atomic mass. The average atomic mass is calculated by the weighted average of all naturally occuring isotopes of said element. The number of protons helps define the element by also being used as the atomic number.
